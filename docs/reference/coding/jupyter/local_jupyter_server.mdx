    (Experimental)
    
    Runs a Jupyter Kernel Gateway server locally.

<b>Parameters:</b>
| Name | Description |
|--|--|
| `ip` | IP address to bind to.<br/><br/>Defaults to "127.0.0.1".<br/><br/>**Type:** `str`<br/><br/>**Default:** '127.0.0.1' |
| `port` | Port to use, if None it automatically selects a port.<br/><br/>Defaults to None.<br/><br/>**Type:** `int \| None`<br/><br/>**Default:** None |
| `token` | Token to use for Jupyter server.<br/><br/>By default will generate a token.<br/><br/>Using None will use no token for authentication.<br/><br/>Defaults to GenerateToken().<br/><br/>**Type:** `str \| GenerateToken`<br/><br/>**Default:** &lt;autogen.coding.jupyter.local_jupyter_server.LocalJupyterServer.GenerateToken object> |
| `log_file` | File for Jupyter Kernel Gateway logs.<br/><br/>Defaults to "jupyter_gateway.log".<br/><br/>**Type:** `str`<br/><br/>**Default:** 'jupyter_gateway.log' |
| `log_level` | Level for Jupyter Kernel Gateway logs.<br/><br/>Defaults to "INFO".<br/><br/>**Type:** `str`<br/><br/>**Default:** 'INFO' |
| `log_max_bytes` | Max logfile size.<br/><br/>Defaults to 1048576.<br/><br/>**Type:** `int`<br/><br/>**Default:** 1048576 |
| `log_backup_count` | Number of backups for rotating log.<br/><br/>Defaults to 3.<br/><br/>**Type:** `int`<br/><br/>**Default:** 3 |
