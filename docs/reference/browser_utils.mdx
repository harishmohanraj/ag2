```python
SimpleTextBrowser(
    start_page: str | None = None,
    viewport_size: int | None = 8192,
    downloads_folder: str | None = None,
    bing_base_url: str = 'https://api.bing.microsoft.com/v7.0/search',
    bing_api_key: str | None = None,
    request_kwargs: dict[str, typing.Any] | None = None
)
```

    (In preview) An extremely simple text-based web browser comparable to Lynx. Suitable for Agentic use.
    
    Initialize the browser with the given parameters.

<b>Parameters:</b>
| Name | Description |
|--|--|
| `start_page` | The initial page to load.<br/><br/>Defaults to None.<br/><br/>**Type:** `str \| None`<br/><br/>**Default:** None |
| `viewport_size` | The number of characters to display per page.<br/><br/>Defaults to 1024 * 8.<br/><br/>**Type:** `int \| None`<br/><br/>**Default:** 8192 |
| `downloads_folder` | The folder to save downloads to.<br/><br/>Defaults to None.<br/><br/>**Type:** `str \| None`<br/><br/>**Default:** None |
| `bing_base_url` | The base URL for Bing searches.<br/><br/>Defaults to "https://api.bing.microsoft.com/v7.0/search".<br/><br/>**Type:** `str`<br/><br/>**Default:** 'https |
| `bing_api_key` | The API key for Bing searches.<br/><br/>Defaults to None.<br/><br/>**Type:** `str \| None`<br/><br/>**Default:** None |
| `request_kwargs` | Additional keyword arguments to pass to the requests library.<br/><br/>Defaults to None.<br/><br/>**Type:** `dict[str, typing.Any] \| None`<br/><br/>**Default:** None |
