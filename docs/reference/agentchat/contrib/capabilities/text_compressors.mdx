```python
LLMLingua(prompt_compressor_kwargs: dict = \{'model_name': 'microsoft/llmlingua-2-bert-base-multilingual-cased-meetingbank', 'use_llmlingua2': True, 'device_map': 'cpu'}, structured_compression: bool = False)
```

    Compresses text messages using LLMLingua for improved efficiency in processing and response generation.
    
    NOTE: The effectiveness of compression and the resultant token savings can vary based on the content of the messages
    and the specific configurations used for the PromptCompressor.

<b>Parameters:</b>
| Name | Description |
|--|--|
| `prompt_compressor_kwargs` | A dictionary of keyword arguments for the PromptCompressor.<br/><br/>Defaults to a dictionary with model_name set to "microsoft/llmlingua-2-bert-base-multilingual-cased-meetingbank", use_llmlingua2 set to True, and device_map set to "cpu".<br/><br/>**Type:** `dict`<br/><br/>**Default:** \{'model_name' |
| `structured_compression` | A flag indicating whether to use structured compression.<br/><br/>If True, the structured_compress_prompt method of the PromptCompressor is used.<br/><br/>Otherwise, the compress_prompt method is used.<br/><br/>Defaults to False.<br/><br/>dictionary.<br/><br/>**Type:** `bool`<br/><br/>**Default:** False |

    relevant information. The compressed text should be stored under the 'compressed_text' key in the dictionary.
